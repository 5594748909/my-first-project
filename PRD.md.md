# 一、文档概述

## 1.1 产品定义

海龟汤是一款**情景推理类休闲互动游戏**，核心玩法为“出题者给出简短汤面（谜面），猜题者通过提问缩小范围、还原完整故事真相（汤底）”。本产品主打轻量化推理、社交互动、碎片化娱乐，兼顾单人闯关与多人联机模式，覆盖全年龄段推理爱好者，打造低门槛、高趣味性的线上推理社交场景。

## 1.2 产品目标

- 打造轻量化、易上手的线上海龟汤玩法，降低传统线下海龟汤的参与门槛，满足用户碎片化推理娱乐需求

- 搭建完善的题库体系与互动机制，提升用户留存与活跃度，打造垂直推理社群

- 实现单人、多人双模式闭环，兼顾独处娱乐与社交互动，拓宽用户使用场景

- 预留内容创作与运营入口，支持题库迭代、活动运营、用户UGC产出，实现产品长期迭代

## 1.3 核心用户定位

|用户层级|用户画像|核心需求|
|---|---|---|
|核心用户|18-35岁，学生、职场白领，喜欢悬疑、推理、剧本杀类内容，热衷碎片化娱乐|优质题库、流畅推理体验、社交互动、成就感反馈|
|潜在用户|12-17岁青少年，休闲玩家，喜欢轻量解谜、朋友互动|简单易上手、趣味性强、无复杂规则、好友联机|
|边缘用户|35岁以上，偶尔参与休闲游戏，偏好轻松无压力玩法|操作简单、内容温和、无恐怖元素|
## 1.4 产品定位与 slogan

**产品定位**：全民级轻量情景推理互动游戏

**核心slogan**：一句话谜面，烧脑还原真相，海龟汤等你来开汤

# 二、产品整体规划

## 2.1 产品架构

产品采用“核心玩法+辅助功能+运营体系”三层架构，分为移动端H5/小程序、APP双端（优先开发小程序/H5），核心模块如下：

- **核心玩法模块**：单人闯关汤、自由开汤、多人联机房、UGC投稿

- **用户体系模块**：注册登录、个人中心、成就系统、等级体系

- **内容管理模块**：题库分类、内容审核、收藏历史、推荐算法

- **社交互动模块**：好友系统、房间聊天、点赞评论、排行榜

- **运营辅助模块**：活动中心、福利礼包、公告系统、反馈入口

## 2.2 核心玩法流程

### 2.2.1 单人模式流程

1. 用户选择汤面分类（经典/悬疑/温情/搞笑等），挑选题目进入游戏

2. 系统展示汤面（谜面），用户通过输入框提问，系统仅回复“是”“否”“无关紧要”

3. 用户可使用提示道具、跳过功能，逐步推理还原汤底

4. 推理成功后，展示完整汤底+解析，解锁成就、获得奖励

### 2.2.2 多人联机模式流程

1. 房主创建房间，选择汤面/自定义汤面，设置房间密码、人数上限

2. 好友/陌生人加入房间，房主担任“汤主”，负责回复玩家提问

3. 玩家轮流提问，汤主实时回复，率先猜出完整汤底的玩家获胜

4. 游戏结束后，展示汤底，可发起重开、更换汤主、切换题库等操作

## 2.3 产品生命周期规划

- **初创期**：搭建核心玩法（单人+基础多人），上线100+优质题库，实现基础功能闭环

- **成长期**：完善社交体系、UGC投稿、成就系统，扩充题库至500+，开展运营活动

- **成熟期**：优化推荐算法、推出主题汤系列、开放赛事功能，搭建用户创作生态

- **迭代期**：根据用户反馈新增玩法、优化体验，拓展跨界合作、IP联动

# 三、核心功能需求

## 3.1 注册登录模块

- **登录方式**：微信授权、手机号验证码登录、游客模式（功能受限，无法保存进度）

- **账号绑定**：支持游客账号绑定手机号/微信，数据迁移，防止进度丢失

- **信息完善**：用户可设置昵称、头像、个性签名，支持修改账号信息

## 3.2 单人闯关模块

### 3.2.1 题库分类

按题材划分：经典海龟汤、温情治愈、悬疑推理、搞笑脑洞、少儿无害（无恐怖元素）；按难度划分：入门、简单、中等、困难、地狱，适配不同水平用户。

### 3.2.2 游戏交互

- 提问框：支持文字输入提问，限制单次提问字数（≤20字），过滤违规词汇

- 回复机制：系统自动判定问题，实时返回“是”“否”“无关紧要”三种结果

- 推理记录：展示历史提问与回复，方便用户梳理思路，支持清空记录

- 辅助功能：提示道具（泄露关键线索）、跳过本题、查看汤底（需消耗道具）

### 3.2.3 结算反馈

闯关成功后展示完整汤底、推理用时、提问次数，发放金币、经验值，解锁对应成就；闯关失败可重新挑战，不扣除道具。

## 3.3 多人联机模块

- **房间创建**：房主可选择系统题库、上传自定义汤面（需审核），设置2-8人房间、公开/私密模式

- **房间管理**：房主拥有踢人、禁言、转让汤主、开始/结束游戏权限

- **实时互动**：房间内公屏聊天、提问发言，汤主专属回复按钮（是/否/无关），全员可见回复记录

- **胜负判定**：玩家完整说出汤底后，汤主确认猜对，判定获胜，发放额外奖励

- **快捷加入**：支持扫码加入、好友邀请、随机匹配公开房间

## 3.4 UGC内容模块

- **投稿功能**：用户可自主创作汤面+汤底，填写分类、难度、标签，提交后台审核

- **审核机制**：机器初审+人工复审，过滤违规、恐怖、低俗、侵权内容，审核通过后上架公共题库

- **创作者权益**：优质投稿标注作者ID，获得金币奖励、专属标识，点赞量高可入选精选题库

## 3.5 用户成长模块

- **等级体系**：通过闯关、联机获胜、投稿获得经验值升级，解锁专属头像框、权限

- **成就系统**：设置各类成就（如“推理大师”“汤面创作者”“联机十连胜”），达成后获得勋章、奖励

- **排行榜**：单日/每周/总榜，分为闯关数、获胜次数、优质投稿三类，展示排名与奖励

## 3.6 辅助功能模块

- **个人中心**：查看闯关进度、历史汤面、收藏题库、好友列表、资产（金币/道具）

- **消息中心**：接收好友申请、系统公告、活动通知、奖励提醒

- **设置中心**：音效开关、消息推送、账号安全、清除缓存、关于我们

- **反馈入口**：用户可提交bug、建议、举报违规内容/用户，后台实时处理

# 四、非功能需求

## 4.1 性能需求

- 页面加载速度：首页、游戏页加载≤2s，联机房间响应≤1s

- 并发支持：支持同时在线1000+用户，多人房间无卡顿、延迟

- 兼容性：适配iOS/Android主流机型，微信小程序、H5端无缝运行

- 稳定性：连续运行48小时无崩溃，断线重连后自动恢复游戏进度

## 4.2 安全与合规需求

- 内容合规：严格过滤恐怖、暴力、低俗、违法内容，未成年人模式屏蔽敏感题材

- 用户隐私：遵守《个人信息保护法》，仅收集必要信息，加密存储用户数据

- 防作弊机制：禁止联机刷分、恶意提问，违规用户禁言、封号处理

- 未成年人保护：开启青少年模式，限制单日游戏时长、消费金额

## 4.3 视觉与交互需求

- 视觉风格：简约暗黑+悬疑氛围感，兼顾温和配色（适配少儿/温和题材），界面简洁无冗余元素

- 交互逻辑：操作极简，一键进入游戏，提问、回复流程顺畅，新手引导清晰易懂

- 动效设计：推理成功、获得奖励时有轻量化动效，提升反馈感

## 4.4 运营需求

- 后台管理：支持题库增删改查、用户管理、内容审核、数据统计、公告发布

- 数据埋点：统计用户活跃度、留存率、题库偏好、联机参与度，支撑产品迭代

- 活动配置：支持限时主题汤、签到福利、闯关挑战赛等活动快速上线
